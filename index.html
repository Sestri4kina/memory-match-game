Memory Match Game

The rules are the following:
  • when you click a cell, it turns red and shows a number
  • when you click the next cell, the previously clicked cell turns blank
  • if a number from the previously clicked cell matches the current one, both cells turn purple
  • the game is over when all the paired cells are matched
Enjoy the Game!
